# vernala-backend

Scraper for African languages

## Datasets
- [ ] https://www.webonary.org/ngiemboon/browse/browse-english/?letter=a&key=en&totalEntries=382&pagenr=1&lang=en

## Commands
- [ ] `uv run main.py` - Run the main script
- [ ] `uv run test.py` - Run the test script
- [ ] `uv run requirements.txt` - Install dependencies
- [ ] `uv add <pkg>` - Add dependency
- [ ] `uv remove <pkg>` - Remove dependency
- [ ] `uv sync` - Install from lockfile
- [ ] `uv python list` - Available Python versions
- [ ] `uv python install X` - Install Python X
- [ ] `uv run <cmd>` - Run inside venv
